## Mumbai :fuelpump: [MATIC token](https://etherscan.io/token/0x7D1AfA7B718fb893dB30A3aBc0Cfc608AaCfeBB0)은 폴리곤의 기본 토큰입니다. 이것은 이더리움의 이더와 비슷합니다. 폴리곤 네트워크와 상호 작용하려면, [MATIC](https://etherscan.io/token/0x7D1AfA7B718fb893dB30A3aBc0Cfc608AaCfeBB0) 토큰이 가스 요금으로 지불되어야 합니다.

- 폴리곤 체인에서 Matic 토큰은 기본 토큰으로 작동합니다. 이것은 사용자가 ETH를 사용하여 이더리움에서 가스 요금을 지불하고 다른 사람에게 ETH를 보내는 것과 같은 방식으로 MATIC으로 가스를 지불할 수 있을 뿐만 아니라 다른 계정으로 MATIC을 보낼 수 있음을 의미합니다.
- 폴리곤 체인의 한 계정에서 다른 계정으로 MATIC 토큰을 보내는 [script](https://gist.github.com/rahuldamodar94/ea3bc4c551e6fc2d318767dcd7e5bffe)의 예입니다.

Matic 토큰을 얻는 것은 정말 쉽습니다. Goerli에 배치된 플라즈마 또는 PoS 컨트랙트를 사용하여 Goerli에서 뭄바이로 MATIC 토큰을 입금할 수 있습니다. 

뭄바이에서 matic 토큰을 얻는 방법 :

- FAUCET: [https://faucet.polygon.technology](https://faucet.polygon.technology/)
    - MATIC 토큰 선택
    - 네트워크 Goerli 또는 Mumbai를 선택
    - 계정 주소를 입력하고 확인

- Goerli ↔ Mumbai간 상호작용하고 싶다면,
    - Maticjs 사용가능 : [https://maticnetwork.github.io/matic.js/](https://maticnetwork.github.io/matic.js/)
    - 또는 새로운 폴리곤 웹 지갑을 사용 �: [https://wallet-dev.polygon.technology/](https://wallet-dev.polygon.technology/)
